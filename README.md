# Med Reminder ISS

# Opening the diagrams:
use [draw.io](https://app.diagrams.net)

## Tech Stack
- Java
- Spring Boot
- PostgreSQL
- Mantine UI

---

## Features
- User Authentication
- Medication Management
  - Add drugs to medication plan
  - Update details about existing drugs of the medication plan
  - Remove drug from the medication plan
  - Medication scheduling
  - Dosage tracking
- Reminder Notifications

---

## Non Functional features
- Easy to use interface
- Data integrity across multiple devices
- Docker containerization
- Linting
